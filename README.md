# 个人网站作品集

这是一个暗黑系视觉风格的静态作品集 MVP，面向视觉设计、平面设计和 UI 设计求职。它不依赖构建工具，直接打开 `index.html` 即可预览，也可以部署到 GitHub Pages、Vercel 或 Netlify。

## 如何替换成你的内容

1. 在 `index.html` 中搜索“你的姓名”“待填写”“hello@example.com”，替换成真实信息。
2. 把真实平面作品和 UI 截图放进 `assets/`，再替换页面中的占位作品区域。
3. 把简历 PDF 命名为 `resume.pdf`，放到 `assets/`，页面里的“下载 PDF 简历”按钮会自动指向它。
4. UI 项目建议写成案例：背景、目标、职责、过程、结果和反思。
5. 平面作品建议按系列展示：品牌视觉、海报、版式、活动物料、包装或社媒图。
6. 暗黑系页面更依赖图片质感，建议优先选择对比强、构图完整、背景干净的作品图。

## 推荐上线方式

- GitHub Pages：适合长期维护，免费稳定。
- Vercel：适合快速预览和自动部署。
- Netlify：适合拖拽上传静态站点。

## 第一版验收清单

- 首页 10 秒内能看懂你是视觉/UI 综合设计方向。
- 至少 2 个 UI 项目可浏览，其中 1 个是完整案例。
- 至少 8-12 张平面作品可浏览，并按系列组织。
- 邮箱、社媒和简历下载入口可用。
- 手机端和电脑端都能正常阅读，没有文字溢出。
